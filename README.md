# TODO ANDROID APP

# Features
- Add/Update/Delete TAsk
- Discard task automatically while experied
- Will show notification 5 minutes earlier of a task
- Automatically sync task from remote server
- Dark mode option

# Technology used
- MVVM Architecture
- Kotlin
- Livedata
- Coroutine
 -Viewbinding
- Navigatinon Components
